# ddb-toolbox-rs
A rust rewrite of [ddb-tools](https://github.com/yuukawahiroshi/ddb-tools) with [GUI](https://github.com/ayatinene/ddb-tools).  

Try the compiled binary files in the latest [Actions](https://github.com/Slidingwall/ddb-toolbox-rs/actions).

I tried to make this thing act like the original Python version. If you want to fix bugs or add features, feel free to open a pull request.
